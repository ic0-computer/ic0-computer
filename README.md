# ic0.computer

A collection of tools and applications for the Internet Computer ecosystem focused on identity management, personal portfolios, and developer utilities.

## Projects

### 🖼️ ic0.computer
Portfolio platform for Internet Computer NFTs and tokens.

**Key Features:**
- Primary account connection via Internet Identity
- Multiple subsidiary account linking with IC wallets
- Unique principal verification system
- Custom principal mining with seed phrase generation
- NFT/token portfolio showcase and sharing
- *Up and Coming:* NFT exchange, portfolio tracking, custom displays

### 🔌 connectr
TypeScript library for Internet Computer wallet integration.

**Features:**
- Seamless connection to Stoic, Plug, and Bitfinity wallets
- Multi-identity management from seed phrases
- Alternative to Connect2IC

### 🛠️ ic0-tools
Utility functions for Internet Computer development.

**Current Tools:**
- Custom avatar generation from principals/account IDs
- Principal to account ID conversion
- Principal generation (random/from seed phrases)

### 🔒 computr extension
Security-focused wallet interface extension (under development).

**Core Features:**
- Non-custodial identity management
- DFX command line instruction generation
- Compatible with air-gapped setups
- Enhanced security through command-line execution
